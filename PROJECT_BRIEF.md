# Financial Reports Platform

## Design Reference
See `data-integrated-layout.html` for the layout we're implementing.

## Key Features to Build
1. Live data strip with real-time market metrics (pull from free APIs)
2. Featured interactive chart (portfolio performance vs benchmarks)
3. Report cards with embedded mini-charts
4. Data-rich sidebar with signals, track record, positioning
5. Clean, readable blog-style report archive

## Tech Stack Preferences
- React
- Data sources: Yahoo Finance (yfinance), NewsAPI, FRED

1. Set up basic project structure
